# 体感大炮射击 - 产品方案

---

## 一、产品概述

**体感大炮射击**基于手机摄像头 + AI 姿态识别的体感射击游戏。用户无需任何外设，只需面对手机前置摄像头，通过手臂动作控制瞄准和发射，击杀屏幕上的僵尸目标获得分数。

**核心卖点**：
- 零外设，手机即玩
- AI 实时识别手部动作，操作直觉自然
- 运动量适中，适合碎片时间健身
- 趣味性强，老少皆宜


---

## 二、核心玩法

### 操作方式

| 动作 | 游戏效果 |
|---|---|
| 伸出手臂移动 | 控制准星瞄准方向 |
| 手臂快速拉回（类似拉弹弓） | 发射炮弹 |

### 游戏流程

```
进入游戏 → 教学提示 → 3秒倒计时 → 60秒对局 → 结算页面
```

### 游戏规则

- 限时 60 秒，尽可能多击杀僵尸
- 僵尸从屏幕上方出现，缓慢向下移动
- 僵尸到达屏幕底部 = 扣一条命（共 3 条命）
- 连续命中有连击加分
- 3 条命用完提前结束

---

## 三、僵尸系统

| 类型 | 特征 | 血量 | 分值 | 出现时机 |
|---|---|---|---|---|
| 普通僵尸 | 绿色，中等大小 | 1 | 100 | 全程 |
| 快速僵尸 | 紫色，较小，移动快 | 1 | 150 | 20秒后 |
| 坦克僵尸 | 棕色，体型大，移动慢 | 2 | 200 | 30秒后 |
| Boss 僵尸 | 红色，巨大，极慢 | 5 | 500 | 50秒后 |

### 难度曲线

- 前 15 秒：轻松热身，只有普通僵尸
- 15-30 秒：加入快速僵尸，节奏加快
- 30-45 秒：加入坦克僵尸，需要多次命中
- 45-60 秒：Boss 出现，全力输出

---

## 四、计分系统

| 事件 | 分数 |
|---|---|
| 击杀普通僵尸 | +100 |
| 击杀快速僵尸 | +150 |
| 击杀坦克僵尸 | +200 |
| 击杀 Boss | +500 |
| 连击加成（连续命中不 miss） | 额外 +50 × 连击数 |

### 结算展示

- 总得分
- 击杀数
- 命中率
- 最高连击

## 五、视觉与交互设计

### 界面布局

```
┌─────────────────────────────┐
│ [摄像头预览]  分数   ⏱倒计时│
│              连击    ❤❤❤   │
├─────────────────────────────┤
│                             │
│    🧟    🧟‍♂️         🧟    │  ← 僵尸从上方出现
│         🧟              │      向下移动
│              ⊕              │  ← 准星（跟随手移动）
│    🧟‍♀️                      │
│                             │
│         💥                  │  ← 命中爆炸特效
│                             │
├─────────────────────────────┤
│         🔫 大炮             │  ← 底部固定
└─────────────────────────────┘
```

### 交互反馈

| 操作 | 反馈 |
|---|---|
| 瞄准移动 | 准星实时跟随手臂 |
| 蓄力（手开始回拉） | 准星缩小 + 颜色变亮 |
| 发射 | 大炮闪光 + 炮弹飞出 + 手机震动 |
| 命中 | 僵尸爆炸 + "命中!" 文字 + 震动 |
| 未命中 | "未中" 灰色提示 |
| 僵尸到达底部 | 屏幕红色闪烁 + 扣心 + 震动 |
| 连击 | 金色放大文字 "x3!" |

### 教学引导

首次进入显示 3 秒教学：
- "伸手瞄准，快速拉回发射"
- 配合简单的手势示意图



## 运动量设计

| 动作 | 运动部位 | 频率 |
|---|---|---|
| 手臂伸出瞄准 | 肩部、上臂 | 持续 |
| 手臂拉回发射 | 肩部、背部 | 每 1-2 秒 |
| 左右移动瞄准 | 核心肌群 | 持续 |

预估每局（60 秒）运动量：
- 手臂伸缩约 40-60 次
- 相当于轻度有氧运动 3-5 分钟

## 六、后续扩展方向

- **多种炮弹**：散弹（范围攻击）、火焰弹（持续伤害）、冰冻弹（减速）
- **道具系统**：炸弹清屏、时间减速、双倍分数
- **关卡模式**：固定波次 + Boss 战 + 剧情
- **双人模式**：两人各控制一门炮，协作击杀
- **排行榜**：全国/城市排名，社交分享
- **皮肤系统**：不同大炮外观、准星样式
- **运动数据**：记录手臂移动距离、消耗卡路里
